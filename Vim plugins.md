# Installation plugins

## 1. Update .vimrc

First add `packloadall` to the `.vimrc` file

## 2. Create the plugins directory

Create the next path directory `~/.vim/pack/plug-ins/start/` 

```bash
mkdir -p ~/.vim/pack/plug-ins/start/
```
